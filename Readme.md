# AssLoader(Chrome extension)

load local .ass file to online video.

https://chrome.google.com/webstore/detail/assloader/kgekkblldbeehilpnmlfmbkgnemcgbkl

## Dependencies

node.js

## Build

commands:

```
cd AssLoaderNpm

npm install

npx webpack
```

## Publish

\AssLoader

*zip, pack to .crx or open in Chrome with development mode.*
